# ZULONG Commercial License

## Why Commercial Licensing?

ZULONG Community Edition is licensed under **AGPL-3.0** with a Plugin Exception.

The AGPL-3.0 requires that if you modify ZULONG and deploy it as a network
service, you must make your modified source code available to users. If this
requirement does not fit your business model, a commercial license is available.

## When Do You Need a Commercial License?

You **need** a commercial license if:

- You want to modify ZULONG framework code and deploy it as a proprietary
  SaaS/cloud service without releasing your modifications
- You want to embed ZULONG into a proprietary product for distribution
- You want access to **Enterprise Edition** modules (advanced memory evolution,
  knowledge graph, dual-brain hot-swap, inference engine, etc.)
- You want priority technical support and SLA guarantees

You **do NOT need** a commercial license if:

- You use the Community Edition as-is (unmodified) for internal purposes
- You develop plugins using the Published Plugin API (covered by the Plugin
  Exception in the LICENSE file)
- You modify ZULONG and comply with AGPL-3.0 by releasing your modifications
- You use ZULONG for academic research or personal projects

## FAQ

**Q: Can I evaluate ZULONG Enterprise before purchasing?**
A: Yes. Contact us for a time-limited evaluation license.

**Q: Are plugins I develop using the Plugin API affected by AGPL?**
A: No. Plugins that use only the Published Plugin API are explicitly exempt
from AGPL obligations under the Plugin Exception clause (see LICENSE file).

**Q: Can I use the Community Edition in a commercial product?**
A: Yes, as long as you comply with AGPL-3.0 requirements (primarily: make
your modified source code available if you deploy it as a network service).

**Q: What if I only use ZULONG internally and never distribute it?**
A: Internal use without network service deployment to external users does
not trigger AGPL-3.0 source code disclosure requirements. However, if
external users interact with your modified version over a network, you
must provide them access to the source code.
