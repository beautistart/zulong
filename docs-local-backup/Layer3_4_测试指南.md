# 视觉模块 Layer 3 & Layer 4 测试指南

## 📋 测试目标

- **Layer 3**: 测试动作分类（MobileNetV4-TSM）和意图识别
- **Layer 4**: 测试手势识别（MediaPipe + EfficientNet）
- **联合测试**: 验证从动作到手势的转换

## 🚀 快速开始

### 方式 1: 快速测试（推荐首次使用）

```bash
python tests/test_layer3_4_quick.py
```

**特点**:
- 10 秒快速验证
- 实时显示状态
- 自动记录最高分数

### 方式 2: 完整测试

```bash
python tests/test_layer3_4_action_gesture.py
```

**特点**:
- 分模块详细测试
- 可选择单独测试 L3 或 L4
- 包含联合测试
- 30 秒/模块

## 📖 测试步骤

### Layer 3: 动作分类测试

1. **启动测试**:
   ```bash
   python tests/test_layer3_4_quick.py
   ```

2. **执行动作**:
   - 站在摄像头前
   - 持续挥手 2-3 秒（左右或上下）
   - 动作幅度适中

3. **预期结果**:
   ```
   Layer 3 (动作分类):
     最大分数：0.850
     状态：✅ 通过
   ```

4. **关键指标**:
   - `action_score > 0.6`: 触发意图识别
   - `action_score > 0.8`: 强烈交互意图
   - 意图类型：`WAVING`, `APPROACHING`, `INTERACTING`

### Layer 4: 手势识别测试

1. **启动测试**:
   ```bash
   python tests/test_layer3_4_quick.py
   ```

2. **执行手势**:
   - **V 字手势**: 食指和中指张开
   - **OK 手势**: 拇指和食指成圈
   - **点赞手势**: 拇指向上
   - **张开手掌**: 五指张开

3. **预期结果**:
   ```
   Layer 4 (手势识别):
     检测到的手势：{'V_SIGN', 'OK', 'THUMBS_UP'}
     状态：✅ 通过
   ```

4. **关键指标**:
   - `gesture_confidence > 0.25`: 检测到手势
   - 鹰眼模式自动触发（5 倍放大）
   - 冷却时间：0.5 秒

### 联合测试

1. **启动测试**:
   ```bash
   python tests/test_layer3_4_action_gesture.py
   ```
   选择选项 `3` 或 `4`

2. **执行流程**:
   - 阶段 1: 持续挥手 2-3 秒
   - 阶段 2: 比出手势

3. **预期结果**:
   ```
   Layer 3 (动作分类): ✅ 通过
   Layer 4 (手势识别): ✅ 通过
   
   总体评价:
     ✅ 两层都工作正常！系统可以正确识别意图和手势！
   ```

'action_num_frames_fast': 16,       # 快流帧数

```python

'digital_zoom_factor': 5.0,         # 鹰眼放大倍数
```

## 🔍 故障排查

### 问题 1: Layer 3 未检测到动作

**可能原因**:
- 动作幅度太小
- 挥手速度太快或太慢
- 光线不足

**解决方案**:
```
1. 增大幅度：手臂完全伸展
2. 保持节奏：1-2 次/秒
3. 改善照明：确保面部和手部清晰
```

### 问题 2: Layer 4 未检测到手势

**可能原因**:
- 手势不清晰
- 距离摄像头太远或太近
- 背景复杂

**解决方案**:
```
1. 标准手势：使用常见手势（V 字、OK、点赞）
2. 合适距离：50-80cm
3. 简化背景：避免杂乱
4. 稳定手势：保持 1-2 秒
```

### 问题 3: 鹰眼模式未触发

**可能原因**:
- 手势置信度低于阈值
- 冷却时间未过

**解决方案**:
```
1. 提高手势质量：确保所有手指可见
2. 等待 0.5 秒后再次尝试
3. 检查日志中的 gesture_confidence
```

## 🧪 测试场景

### 场景 1: 挥手触发意图

```
用户行为：站在摄像头前，持续挥手 2-3 秒
预期结果：
  - action_score > 0.6
  - intent_type = "WAVING" 或 "INTERACTING"
  - 发布 INTERACTION_TRIGGER 事件
```

### 场景 2: V 字手势识别

```
用户行为：比出 V 字手势，保持 1 秒
预期结果：
  - gesture_type = "V_SIGN"
  - gesture_confidence > 0.5
  - 鹰眼模式激活
```

### 场景 3: 连续手势切换

```
用户行为：V 字 → OK → 点赞（每个间隔 1 秒）
预期结果：
  - 依次检测到三种手势
  - 每次切换响应时间 < 0.5s
  - 无遗漏
```

## 🎓 最佳实践

1. **测试前准备**:
   - 确保摄像头清洁
   - 调整合适的光线
   - 选择简洁背景

2. **动作执行**:
   - 自然、流畅
   - 幅度适中
   - 保持稳定

3. **手势执行**:
   - 标准手势
   - 手指完全伸展
   - 面向摄像头

4. **结果分析**:
   - 关注趋势而非单帧
   - 结合多个测试场景
   - 记录异常情况
