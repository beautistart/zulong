# 复盘机制流程图

## 完整复盘流程

```mermaid
graph TB
    Start([用户输入]) --> KeywordCheck{检测关键词}
    
    KeywordCheck -->|"启动复盘"| InitReview[初始化复盘]
    KeywordCheck -->|"快速/深度复盘"| DirectStart[直接启动对应模式]
    KeywordCheck -->|"结束复盘"| EndReview[结束复盘]
    KeywordCheck -->|其他 | NormalFlow[正常对话流程]
    
    InitReview --> ModeSelect[进入模式选择阶段]
    ModeSelect --> UserChoice{用户选择}
    
    UserChoice -->|"快速复盘"| QuickMode[快速复盘模式]
    UserChoice -->|"深度复盘"| DeepMode[深度复盘模式]
    UserChoice -->|"退出"| CancelReview[取消复盘]
    
    DirectStart --> ModeCheck{检查模式}
    ModeCheck -->|"快速"| QuickMode
    ModeCheck -->|"深度"| DeepMode
    
    QuickMode --> CreateBuffer[创建临时缓冲区]
    DeepMode --> CreateBuffer
    
    CreateBuffer --> EnterDialog[进入对话进行阶段]
    
    EnterDialog --> UserInput[用户自由对话]
    UserInput --> RecordBuffer[记录到缓冲区]
    RecordBuffer --> L2Response[L2 正常回复]
    L2Response --> CheckEnd{是否结束？}
    
    CheckEnd -->|"结束复盘"| ExtractExp[提取经验]
    CheckEnd -->|继续对话 | UserInput
    
    ExtractExp --> CallExtractor[调用 ExperienceExtractor]
    CallExtractor --> AnalyzeData[分析缓冲区数据]
    AnalyzeData --> GenerateExp[生成经验草案]
    GenerateExp --> EnterConfirm[进入经验确认阶段]
    
    EnterConfirm --> ShowExp[展示经验给用户]
    ShowExp --> UserConfirm{用户确认？}
    
    UserConfirm -->|"确认"| SaveExp[保存到经验库]
    UserConfirm -->|"修改"| RestartDialog[重新开始对话]
    UserConfirm -->|"退出"| CancelExp[取消保存]
    
    SaveExp --> Complete([复盘完成])
    CancelExp --> ExitReview([退出复盘])
    RestartDialog --> EnterDialog
    
    CancelReview --> ExitReview
    EndReview --> ExtractExp
```

## 三阶段状态机

```mermaid
stateDiagram-v2
    [*] --> ModeSelecting: 启动复盘
    
    state ModeSelecting {
        [*] --> WaitingChoice
        WaitingChoice --> QuickSelected: 用户选择"快速"
        WaitingChoice --> DeepSelected: 用户选择"深度"
        WaitingChoice --> Cancelled: 用户选择"退出"
    }
    
    ModeSelecting --> ReviewActive: 选择模式
    
    state ReviewActive {
        [*] --> DialogInProgress
        DialogInProgress --> RecordingUserInput: 用户输入
        RecordingUserInput --> L2Responding: L2 回复
        L2Responding --> DialogInProgress
        DialogInProgress --> EndRequested: 用户说"结束复盘"
    }
    
    ReviewActive --> ExperienceConfirming: 结束复盘
    
    state ExperienceConfirming {
        [*] --> ShowingExperiences
        ShowingExperiences --> WaitingConfirm: 展示经验
        WaitingConfirm --> Confirmed: 用户确认
        WaitingConfirm --> ModifyRequested: 用户修改
        WaitingConfirm --> ExitRequested: 用户退出
    }
    
    ExperienceConfirming --> Completed: 确认保存
    ExperienceConfirming --> ReviewActive: 修改重新对话
    ExperienceConfirming --> Cancelled: 退出
    
    ModeSelecting --> Cancelled: 取消
    Cancelled --> [*]
    Completed --> [*]
```

## 组件交互流程

```mermaid
sequenceDiagram
    participant User as 用户
    participant Gatekeeper as L1-B Gatekeeper
    participant ReviewTrigger as ReviewTrigger
    participant ReplayIntegration as ReplayIntegration
    participant Buffer as ReviewBuffer
    participant Extractor as ExperienceExtractor
    participant L2 as L2 推理引擎
    
    User->>Gatekeeper: "启动复盘"
    Gatekeeper->>Gatekeeper: 检测关键词
    Gatekeeper->>ReviewTrigger: trigger_user_active()
    
    ReviewTrigger->>ReviewTrigger: 发布 REPLAY_TRIGGERED 事件
    ReviewTrigger->>ReplayIntegration: on_replay_triggered()
    
    ReplayIntegration->>ReplayIntegration: 进入模式选择阶段
    ReplayIntegration->>L2: 发布模式选择提示
    
    L2->>User: "请选择：快速/深度复盘"
    
    User->>Gatekeeper: "快速复盘"
    Gatekeeper->>ReplayIntegration: handle_mode_selection()
    
    ReplayIntegration->>ReplayIntegration: 进入对话阶段
    ReplayIntegration->>Buffer: create_buffer()
    ReplayIntegration->>L2: 发布对话提示
    
    L2->>User: "已进入快速复盘模式，请自由对话"
    
    User->>Gatekeeper: "今天学习了复盘机制"
    Gatekeeper->>Buffer: add_user_input()
    Gatekeeper->>L2: 转发输入
    
    L2->>L2: 生成回复
    L2->>User: "复盘机制是..."
    
    User->>Gatekeeper: "结束复盘"
    Gatekeeper->>ReplayIntegration: handle_end_review()
    
    ReplayIntegration->>Buffer: export_for_analysis()
    ReplayIntegration->>Extractor: extract_from_buffer()
    
    Extractor->>Extractor: 构建 L2 提示词
    Extractor->>L2: 调用分析
    L2->>Extractor: 返回 JSON 经验
    
    Extractor->>ReplayIntegration: 结构化经验数据
    ReplayIntegration->>ReplayIntegration: 进入经验确认阶段
    ReplayIntegration->>L2: 发布经验确认提示
    
    L2->>User: "已生成 3 条经验，请确认"
    
    User->>Gatekeeper: "确认"
    Gatekeeper->>ReplayIntegration: handle_experience_confirmation()
    
    ReplayIntegration->>ReplayIntegration: 保存到经验库
    ReplayIntegration->>ReplayIntegration: 退出复盘模式
    ReplayIntegration->>L2: 发布完成提示
    
    L2->>User: "✅ 复盘完成，已保存 3 条经验"
```

## 关键决策点

```mermaid
graph TD
    Start([用户输入]) --> Check1{是否包含"复盘"？}
    
    Check1 -->|是 | Check2{是否已激活复盘模式？}
    Check1 -->|否 | Normal[正常处理]
    
    Check2 -->|否 | Check3{是否为启动指令？}
    Check2 -->|是 | Check4{当前阶段？}
    
    Check3 -->|"启动复盘"| TriggerReview[触发 ReviewTrigger]
    Check3 -->|"快速/深度复盘"| DirectStart[直接启动]
    Check3 -->|其他 | Ignore[忽略]
    
    Check4 -->|模式选择 | HandleModeSelect[处理模式选择]
    Check4 -->|对话进行 | Check5{是否为结束指令？}
    Check4 -->|经验确认 | HandleConfirm[处理确认]
    
    Check5 -->|是 | ExtractExperience[提取经验]
    Check5 -->|否 | RecordAndForward[记录并转发给 L2]
    
    TriggerReview --> End([结束])
    DirectStart --> End
    HandleModeSelect --> End
    ExtractExperience --> End
    HandleConfirm --> End
    RecordAndForward --> End
    Normal --> End
    Ignore --> End
```
