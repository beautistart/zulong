## 📝 完整测试步骤

### 测试 1: Layer 1 人体检测
1. 站在摄像头前，距离 1-2 米
2. 确保全身或上半身在画面中
3. 观察绿色人体检测框
4. **预期**: 绿色框框住人体，显示置信度 > 0.5

### 测试 2: Layer 2 运动检测
1. 保持站在摄像头前
2. 挥动手臂（大幅度）
3. 观察左上角 "Motion: XXXXpx"
4. **预期**: 运动像素 > 1000px，显示绿色

### 测试 3: Layer 3 意图识别
1. **继续挥手 2-3 秒**（关键！）
2. 不要停下，持续挥动
3. 观察 L3 显示
4. **预期**: 
   - 动作分数从 0.1 逐渐上升到 > 0.6
   - 意图类型：WAVING（挥手）
   - 颜色从红色→黄色→绿色

### 测试 4: Layer 4 手势识别
1. 停止挥手，保持静止
2. 比出以下手势（每个保持 2 秒）：
   - ✋ **张开手掌** (Open_Palm)
   - ✌️ **V 字手势** (Victory_Sign)
   - 👌 **OK 手势** (OK_Gesture)
   - ✊ **拳头** (Closed_Fist)
3. 观察 L4 显示
4. **预期**: 显示手势名称和置信度

### 测试 5: L0-L4 完整联动
1. 坐在摄像头前静止 3 秒
2. 挥动手臂 2-3 秒
3. 继续挥手并比出 V 字手势
4. 观察控制台事件输出
5. **预期**: 
   - 收到 SENSOR_VISION_STATE 事件
   - 收到 INTERACTION_TRIGGER 事件

## ❗ 常见问题

### Q1: Layer 1 检测不到人
- 检查光线是否充足
- 确保人体在画面中占比适中
- 调整摄像头角度

### Q2: Layer 2 检测不到运动
- 挥手幅度要大
- 运动阈值已设为 100px（很灵敏）
- 检查是否站得太远

### Q3: Layer 3 分数上不去
- **必须持续挥手 2-3 秒**
- 动作分类需要时间累积分数
- 不要挥一下就停

### Q4: Layer 4 手势不识别
- 手势要清晰标准
- 手掌正对摄像头
- 距离摄像头 50-80cm 最佳
- 置信度阈值已设为 0.3（很灵敏）
