## 🎯 保护效果

### AGPL-3.0 保护

```
✅ 企业SaaS使用 → 必须开源全部代码
✅ 嵌入闭源产品 → 必须购买商业授权
✅ 提供商业谈判筹码
✅ 大公司法务会谨慎评估
```

### MIT 开放

```
✅ 开发者可自由使用前端
✅ 鼓励集成到其他项目
✅ 降低使用门槛
✅ 扩大影响力
```
