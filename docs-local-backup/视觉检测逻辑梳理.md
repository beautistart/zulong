# 祖龙视觉系统 - 四层检测逻辑梳理

## 📊 整体架构

```
摄像头输入 (640x480 @ 30fps)
     ↓
┌────────────────────────────────────────────────────────────┐
│           OptimizedVisionProcessor                         │
│  (人体锚点驱动策略 | TSD v1.7 4.4 感知预处理)                │
└────────────────────────────────────────────────────────────┘
     ↓
  Layer 1 → Layer 2 → Layer 3 → Layer 4
  (人体)    (运动)    (意图)    (手势)
```

---

## 🔍 Layer 1: 人体检测 (YOLO-Nano)

### 触发条件
- **每帧都执行**，但 YOLO 推理**每 5 帧执行一次**

### 核心逻辑
```python
# 文件：zulong/l1c/optimized_vision_processor.py
# 方法：_layer1_human_detection()

1. 帧计数器 +1
2. 检查：当前帧 - 上次 YOLO 推理帧 < 5 ?
   - 是 → 跳过 YOLO，使用 shared_memory 中的上一帧结果
   - 否 → 执行 YOLO 推理
3. YOLO 推理：
   - 输入：BGR 帧 (640x480)
   - 输出：人体检测框 [x1, y1, x2, y2] + 置信度
4. 更新 shared_memory['human_bbox']
```

### 性能优化
- **推理频率**: 1/5 (每 5 帧执行一次 YOLO)
- **缓存策略**: 使用 shared_memory 缓存上一次检测结果
- **模型**: YOLOv8n (已量化，CPU 运行)

### 输出
- ✅ 检测到人：返回 `[[x1, y1, x2, y2]]`
- ❌ 未检测到人：返回 `[]`

---

## 👆 Layer 2: ROI 运动检测 (光流法)

### 触发条件
- **每帧都执行** (前提是 Layer 1 检测到人)

### 核心逻辑
```python
# 文件：zulong/l1c/optimized_vision_processor.py
# 方法：_layer2_roi_motion_detection()

1. 提取 ROI 区域：
   - 基于 Layer 1 的 bbox
   - 向外扩展 50px (包含手部活动区域)
   
2. 帧间差分：
   - 当前帧 ROI vs 帧缓冲区前一帧 ROI
   - 转为灰度图：cv2.cvtColor(BGR → GRAY)
   
3. ROI 增益放大：
   - 应用 cv2.convertScaleAbs(alpha=3.0, beta=0)
   - 放大 3 倍以增强微小运动
   
4. 二值化阈值：
   - 阈值：100 (可调)
   - 方法：cv2.threshold(diff, 100, 255, THRESH_BINARY)
   
5. 计算运动像素：
   - motion_pixels = cv2.countNonZero(thresh)
   
6. 判断：
   - motion_pixels > 100 ? 检测到运动 : 无运动
```

### 性能瓶颈
- **灰度转换**: `cv2.cvtColor` 较慢
- **帧缓冲读取**: `self.frame_buffer[-2]` 需要拷贝

### 输出
- ✅ 检测到运动：`(True, motion_pixels)`
- ❌ 无运动：`(False, 0)` → **提前返回，不执行 Layer 3/4**

---

## 🧠 Layer 3: 动作分类 (MobileNetV4-TSM)

### 触发条件
- **每 2 帧执行一次** (偶数帧)
- 前提是 Layer 2 检测到运动

### 核心逻辑
```python
# 文件：zulong/l1c/optimized_vision_processor.py
# 方法：_layer3_action_classification()

1. 帧计数器检查：
   - frame_counter % 2 == 0 ? 执行 : 使用缓存结果
   
2. 执行动作分类 (仅偶数帧):
   - 输入：当前帧 + 帧缓冲区 (8 帧 Slow 流 + 16 帧 Fast 流)
   - 模型：MobileNetV4-TSM (双流架构)
   - 输出：2560 维特征向量
   
3. 规则分类：
   - 计算特征能量：energy = mean(|features|)
   - 计算特征差异：diff = mean(|fast_feat - slow_feat|)
   
4. 意图判断：
   if energy > 0.05 and diff > 0.015:
       → "WAVING" (挥手)
   elif energy > 0.02 and diff <= 0.015:
       → "APPROACHING" (靠近)
   elif energy <= 0.02 and diff > 0.008:
       → "GAZING" (注视)
   else:
       → "STILL" (静止)
   
5. 计算置信度：
   - 挥手：score = 0.75 + (energy + diff) * 0.25
   - 靠近：score = 0.7 + energy * 0.3
   - 注视：score = 0.7 + diff * 0.3
   - 静止：score = 0.7 (默认)
```

### 输出
- `action_score`: 0.0-1.0 (置信度)
- `intent_type`: "WAVING" | "APPROACHING" | "GAZING" | "STILL"

### 后续流程
```python
if action_score < 0.6:
    → NO_INTENT (返回，不执行 Layer 4)
    
elif action_score < 0.8:
    → SILENT_WATCH (发布 INTERACTION_TRIGGER 事件，排队模式)
    
else:
    → INTERACT_REQUEST (执行 Layer 4)
```

## 🦅 Layer 4: 鹰眼模式 (Digital Zoom + EfficientNet)

### 触发条件
- Layer 3 的 `action_score >= 0.8`
- 距离上次执行 > 0.5 秒 (cooldown)

### 核心逻辑
```python
# 文件：zulong/l1c/optimized_vision_processor.py
# 方法：_layer4_eagle_eye_mode()

1. Cooldown 检查：
   - timestamp - last_eagle_eye_time < 0.5 ?
   - 是 → 跳过，返回 None
   
2. Digital Zoom (数字变焦):
   - 基于 Layer 1 的 bbox
   - 裁剪 ROI 区域
   - 放大 5.0 倍
   - 双三次插值：cv2.resize(roi, (224, 224), interpolation=INTER_CUBIC)
   
3. 手势识别：
   - 模型：EfficientNet-B0 (已预训练)
   - 输入：224x224 裁剪图像
   - 输出：手势类别 + 置信度
   
4. 手势类别：
   - ✋ OPEN_PALM (张开手掌)
   - ✌️ VICTORY (V 字手势)
   - 👌 OK (OK 手势)
   - ✊ FIST (握拳)
   
5. 判断：
   - confidence > 0.25 ? 识别成功 : 识别失败
```

### 输出
- ✅ 识别成功：`{'gesture': 'VICTORY', 'confidence': 0.85}`
- ❌ 识别失败：`None`

### 事件发布
```python
if gesture_result:
    → 发布 INTERACTION_TRIGGER 事件 (HIGH 优先级，中断模式)
    → payload: {
        'gesture': 'VICTORY',
        'gesture_confidence': 0.85,
        'intent_type': 'WAVING',
        'route_mode': 'interrupt',
        'layer': 'L4',
        'eagle_eye_active': True
      }
```

## 📈 完整流程图

```
摄像头帧输入
    ↓
┌───────────────────────────────────────┐
│ Layer 1: 人体检测 (每帧，YOLO 1/5)     │
└───────────────────────────────────────┘
    ↓
    ├─ ❌ 未检测到人 → 返回，state='NO_HUMAN'
    ↓
    ├─ ✅ 检测到人 → shared_memory['human_bbox'] = [x1,y1,x2,y2]
    ↓
┌───────────────────────────────────────┐
│ Layer 2: ROI 运动检测 (每帧)            │
│ 1. 提取 ROI (bbox + 50px margin)       │
│ 2. 灰度转换 + 增益放大 (α=3.0)         │
│ 3. 帧间差分 + 二值化 (阈值=100)        │
│ 4. 计算运动像素                        │
└───────────────────────────────────────┘
    ↓
    ├─ ❌ motion_pixels < 100 → 返回，state='IDLE'
    ↓
    ├─ ✅ motion_pixels >= 100 → 发布 SENSOR_VISION_STATE 事件
    ↓
┌───────────────────────────────────────┐
│ Layer 3: 动作分类 (每 2 帧，偶数帧执行)  │
│ 1. 提取 Slow 流 (8 帧) + Fast 流 (16 帧)  │
│ 2. MobileNetV4-TSM 推理 → 2560 维特征   │
│ 3. 计算 energy 和 diff                 │
│ 4. 规则分类 → WAVING/APPROACHING/GAZING│
│ 5. 计算置信度 score (0.0-1.0)          │
└───────────────────────────────────────┘
    ↓
    ├─ score < 0.6 → 返回，state='NO_INTENT'
    ↓
    ├─ 0.6 <= score < 0.8 → state='SILENT_WATCH'
    │   → 发布 INTERACTION_TRIGGER (排队模式)
    ↓
    ├─ score >= 0.8 → state='INTERACT_REQUEST'
    ↓
┌───────────────────────────────────────┐
│ Layer 4: 鹰眼模式 (cooldown 0.5s)      │
│ 1. Digital Zoom (5.0x, 224x224)       │
│ 2. EfficientNet 手势识别               │
│ 3. 判断手势类别 (OPEN_PALM/VICTORY/OK)│
└───────────────────────────────────────┘
    ↓
    ├─ ❌ confidence < 0.25 → 返回
    ↓
    ├─ ✅ confidence >= 0.25 → state='EAGLE_EYE_ACTIVE'
    │   → 发布 INTERACTION_TRIGGER (中断模式)
```

## ⚡ 性能分析

### 当前 FPS: ~11 FPS

| Layer | 执行频率 | 耗时估算 | 瓶颈分析 |
|-------|---------|---------|---------|
| L1 (YOLO) | 1/5 | ~50ms | YOLO 推理本身不慢，但**每帧都调用**有开销 |
| L2 (ROI) | 每帧 | ~30ms | **灰度转换** + 帧缓冲拷贝 |
| L3 (动作) | 1/2 | ~40ms | MobileNet 推理 + 帧缓冲区读取 |
| L4 (手势) | 触发式 | ~20ms | Digital Zoom + EfficientNet |

### 主要瓶颈
1. **帧缓冲区操作频繁**: 每帧都要读取 `frame_buffer[-2]` 和 `frame_buffer[-8:]`
2. **灰度转换**: Layer 2 每帧都调用 `cv2.cvtColor`
3. **同步处理**: `_process_frame_async` 虽然是 async，但内部是**顺序执行**

### 优化建议
1. **减少帧缓冲依赖**: Layer 2 可以使用 IIR 滤波器替代帧间差分
2. **缓存灰度图**: 避免重复转换
3. **并行处理**: Layer 2 和 Layer 3 可以并行执行

## 🎯 事件发布机制

### SENSOR_VISION_STATE (Layer 2 触发)
```python
{
    'type': EventType.SENSOR_VISION_STATE,
    'priority': EventPriority.NORMAL,
    'source': 'OptimizedVisionProcessor/Layer2',
    'payload': {
        'motion_pixels': 1234,
        'human_bbox': [x1, y1, x2, y2],
        'layer': 'L2',
        'motion_detected': True
    }
}
```

### INTERACTION_TRIGGER (Layer 3 触发 - 静默注意)
```python
{
    'type': EventType.INTERACTION_TRIGGER,
    'priority': EventPriority.NORMAL,
    'source': 'OptimizedVisionProcessor/Layer3',
    'payload': {
        'intent_type': 'WAVING',
        'intent_confidence': 0.72,
        'route_mode': 'queue',  # 排队模式
        'layer': 'L3'
    }
}
```

### INTERACTION_TRIGGER (Layer 4 触发 - 交互请求)
```python
{
    'type': EventType.INTERACTION_TRIGGER,
    'priority': EventPriority.HIGH,
    'source': 'OptimizedVisionProcessor/Layer4',
    'payload': {
        'intent_type': 'WAVING',
        'gesture': 'VICTORY',
        'gesture_confidence': 0.85,
        'route_mode': 'interrupt',  # 中断模式
        'layer': 'L4',
        'eagle_eye_active': True
    }
}
```

## 📝 关键配置参数

```python
# zulong/l1c/optimized_vision_processor.py

_config = {
    # YOLO 频率控制
    'yolo_inference_frequency': 5,  # 每 5 帧执行一次
    
    # ROI 增益

    'gesture_conf_threshold': 0.25, # 手势置信度阈值

}
```
