### 问题 1: dependency_analyze 参数处理错误

**现象**: `"'str' object has no attribute 'get'"`

**位置**: `zulong/skill_packs/packs/complex_task/planning_tools.py`

**原因**: 工具内部对参数的类型假设错误

**建议修复**:
```python
# 在 DependencyAnalyzeTool.execute() 中
# 添加类型检查和转换
subtasks = request.parameters.get("subtasks", [])
if isinstance(subtasks, str):
    subtasks = json.loads(subtasks)
```

### 问题 2: priority_rank 参数处理错误

**现象**: 同问题 1

**位置**: 同上

**建议修复**: 同问题 1

**现象**: 
- 任务分解期望 `goal` 参数
