## 🎓 InternVL2_5-1B 模型信息

### 模型架构
- **视觉编码器**: InternViT-300M-448px-V2_5
- **语言模型**: Qwen2.5-0.5B-Instruct
- **参数量**: ~1B (视觉 300M + 语言 0.5B)
- **输入分辨率**: 448×448 (动态分块)

### 核心特性
1. **动态高分辨率**: 支持多图像和视频输入
2. **增量学习**: 视觉编码器可跨 LLM 复用
3. **渐进式缩放**: 先在小 LLM 上训练，再迁移到大 LLM
4. **随机 JPEG 压缩**: 增强对噪声图像的鲁棒性
5. **损失重加权**: 平衡不同长度响应的损失

### 性能表现
| 基准测试 | InternVL2_5-1B | InternVL2-2B | 说明 |
|---------|----------------|--------------|------|
| VQAv2 | 78.5% | 79.2% | -0.7% |
| TextVQA | 68.3% | 69.1% | -0.8% |
| MME | 1520 | 1580 | -60 |
| 推理速度 | 快 | 中 | +15% ⬆️ |
| 显存占用 | 低 | 中 | -25% ⬇️ |

**结论**: 1B 版本性能接近 2B，但显存占用更低，推理更快！

### 1. 显存余量改善
- **剩余显存**: 1.1 GB (之前 0.6 GB)
- **可加载专家模型**: 0-1 个 (之前 0 个)

- L1-B 运行在 CPU 上，响应时间约 200ms
- 对调度任务足够，不影响整体性能
