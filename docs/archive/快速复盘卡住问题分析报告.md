## 💥 问题根源分析

### 问题 1: 事件循环冲突

**场景**: Gatekeeper 在**已有事件循环**的线程中调用 `_handle_user_active_review()`

```python
# scheduler_gatekeeper.py 第 644 行
def _forward_to_replay_integration_immediate(self, text, priority):
    replay_integration = get_replay_integration()
    
    if review_type == 'quick':
        # ❌ 直接同步调用 (在当前线程的事件循环中)
        replay_integration._handle_user_active_review({
            'user_input': text,
            'review_type': 'quick'
        })
```

**冲突**:
- Gatekeeper 线程：**已有事件循环** (从 bootstrap 启动)
- `_get_recent_context()`: **创建新事件循环** 并尝试 `run_until_complete`
- **结果**: 在已有事件循环的线程中，`run_until_complete` 会**阻塞等待**，但异步任务无法完成!

---

### 问题 2: 异步任务无法完成

**原理**:

```python
# 在已有事件循环的线程中
loop = asyncio.new_event_loop()
asyncio.set_event_loop(loop)

# ❌ 这会阻塞当前线程
result = loop.run_until_complete(async_task)

# 但 async_task 需要事件循环来运行
# 而当前线程被阻塞了，事件循环无法调度任务
# → 死锁！
```

**日志证据**:
- ✅ "已创建临时缓冲区" (同步代码，立即执行)
- ❌ 之后没有任何日志 (`_get_recent_context()` 卡住，无法继续)

---

    async def fetch_data():
        pool = await SharedMemoryPool.get_instance()

    with concurrent.futures.ThreadPoolExecutor() as executor:

            lambda: asyncio.new_event_loop().run_until_complete(fetch_data())

    return result

async def _get_recent_context(self, minutes: int = 30) -> Dict[str, Any]:
    """🔥 修复：异步获取上下文数据"""
    try:
        # ✅ 直接使用当前事件循环
        pool = await SharedMemoryPool.get_instance()
        
        recent_texts = await pool.get_recent(time_window_sec=minutes*60)
        
        logger.info(f"[ReplayIntegration] 从共享池获取到 {len(recent_texts)} 条数据")
        
        conversations = []
        for item in recent_texts:
            if hasattr(item, 'data'):
                conversations.append({
                    'trace_id': item.trace_id if hasattr(item, 'trace_id') else None,
                    'text': item.data.get('text', '') if hasattr(item, 'data') else '',
                    'timestamp': item.timestamp if hasattr(item, 'timestamp') else None,
                    'metadata': item.metadata if hasattr(item, 'metadata') else {}
                })
        
        logger.info(f"[ReplayIntegration] 构建了 {len(conversations)} 条对话记录")
        
        return {
            'conversations': conversations,
            'time_range': {
                'start': (datetime.utcnow() - timedelta(minutes=minutes)).isoformat(),
                'end': datetime.utcnow().isoformat()
            }
        }
        
    except Exception as e:
        logger.error(f"[ReplayIntegration] 获取上下文数据失败：{e}", exc_info=True)
        return {'conversations': [], 'time_range': {}}

## 🎓 总结

**核心问题**:
- ❌ `_get_recent_context()` 在已有事件循环的线程中创建新事件循环
- ❌ `run_until_complete` 阻塞当前线程，导致异步任务无法完成

**修复方案**:
- ✅ 将 `_get_recent_context()` 改为纯异步方法
- ✅ 将 `_handle_user_active_review()` 改为异步方法
- ✅ 将 `_handle_quick_review()` 改为异步方法
- ✅ Gatekeeper 使用 `asyncio.create_task()` 调用
