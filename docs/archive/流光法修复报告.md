# 流光法运动检测失败原因分析与修复报告

**日期**: 2026-03-25  
**版本**: TSD v1.7  
**状态**: ✅ 已修复

---

## 📋 问题概述

流光法运动检测器无法正常工作，系统**定时获取画面**而不是**检测到画面变动时获取画面**，违反了 TSD v1.7 的事件驱动原则。

---

## 🔍 根本原因分析

### ❌ 问题 1：`set_baseline` 方法缺失

**位置**: `zulong/l1a/motion_detector.py`

**现象**:
- `vision_processor.py` 第 218 行调用 `self.motion_detector.set_baseline(frame)`
- 但 `OpticalFlowMotionDetector` 和 `AdaptiveMotionDetector` 都没有这个方法

**影响**:
- 第一帧基准帧设置逻辑混乱
- 导致后续运动检测无法正确初始化

**修复方案**:
```python
def set_baseline(self, frame: np.ndarray):
    """
    手动设置基准帧（用于外部强制初始化）
    
    Args:
        frame: 当前帧 (BGR 格式)
    
    TSD v1.7 对应:
    - 4.4 感知预处理：基准帧初始化
    """
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    self.baseline_gray = gray
    self.baseline_frame = frame.copy()
    self.baseline_set_time = time.time()
    self.prev_gray = gray
    self.prev_frame = frame.copy()
    self._consecutive_motion_count = 0
    logger.info(f"👁️ 手动设置基准帧 (亮度={np.mean(gray):.2f})")
```

### ❌ 问题 2：缺少 `time` 模块导入

**位置**: `zulong/l1a/motion_detector.py`

**现象**:
- `set_baseline` 方法中调用 `time.time()`
- 但文件头部没有导入 `time` 模块

**修复方案**:
```python
import cv2
import numpy as np
import logging
import time  # ✅ 新增

```

### ❌ 问题 3：第一帧处理逻辑错误

**位置**: `zulong/l1a/vision_processor.py`

**现象**:
```python
# ❌ 原代码
if not self.baseline_initialized:
    # 第一帧会在 detect_and_record 中自动设为基准帧
    self.baseline_initialized = True
    logger.debug("👁️ 第一帧将自动设为基准帧")
```

**问题**:
- 依赖 `detect_and_record` 自动设置基准帧
- 但 `vision_processor` 已经尝试手动调用 `set_baseline`
- 逻辑不一致

**修复方案**:
```python
# ✅ 修复后
if not self.baseline_initialized and frame is not None and self.motion_detector is not None:
    logger.info("👁️ 使用启动后的第一帧作为基准帧 (避免重复打开摄像头)")
    # 使用新增的 set_baseline 方法手动设置基准帧
    self.motion_detector.flow_detector.set_baseline(frame)
    self.baseline_initialized = True
    logger.info("✅ 基准帧设置完成")
    return  # 第一帧只用于初始化，不触发运动检测
```

---

### ❌ 问题 4：`get_baseline_frame` 返回错误

**位置**: `zulong/l1a/motion_detector.py`

**现象**:
```python
# ❌ 原代码
def get_baseline_frame(self) -> Optional[np.ndarray]:
    return self.prev_frame.copy() if self.prev_frame is not None else None
```

**问题**:
- 应该返回 `baseline_frame` 而不是 `prev_frame`
- `prev_frame` 是连续帧，`baseline_frame` 才是静态基准

**修复方案**:
```python
# ✅ 修复后
def get_baseline_frame(self) -> Optional[np.ndarray]:
    return self.baseline_frame.copy() if self.baseline_frame is not None else None
```
