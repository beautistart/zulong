## 🎯 设计理念

### 极简主义
1. **核心优先**: 只保留最核心的身份认知
2. **减少干扰**: 避免过多规则导致模型困惑
3. **自然学习**: 让模型通过对话历史自然理解人称

### 信任模型
1. **少即是多**: 相信模型的理解能力
2. **示例胜过规则**: 通过实际对话示例学习
3. **灵活性**: 不限制模型的表达方式
