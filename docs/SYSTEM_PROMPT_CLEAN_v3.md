## 🎯 设计理念

### 极简主义 3.0 - 纯净版
1. **去除所有装饰**: 只保留必要信息
2. **直接定义**: 不添加任何修饰语
3. **信任模型**: 完全依赖模型的理解能力
4. **最少干预**: 让模型自然表现

### 核心原则
- ✅ **必要**: 只保留不可或缺的信息
- ✅ **清晰**: 表达直接、无歧义
- ✅ **简洁**: 字数最少化
- ✅ **有效**: 保证功能完整性
