## 🔍 故障排查

### 问题 1: 摄像头无法初始化

**错误**:
```
❌ 摄像头初始化失败
```

**解决**:
1. 检查摄像头是否被其他程序占用 (关闭 Zoom、Teams 等)
2. 修改设备索引:
   ```python
   # 在脚本第 50 行附近
   camera = CameraDevice(device_index=0)  # 改为 0 试试
   ```

---

### 问题 2: 未检测到人脸

**错误**:
```
⚠️  未检测到人脸
```

**解决**:
1. 确保光线充足，避免逆光
2. 正对摄像头，距离 30-50cm
3. 摘掉口罩、眼镜 (如果可能)
4. 降低检测阈值:
   ```python
   # 在 l1c_vision_silent_plugin.py 第 86 行
   FACE_CONFIDENCE_THRESHOLD = 0.7  # 从 0.8 改为 0.7
   ```

---

### 问题 3: 麦克风无声音

**错误**:
```
⚠️  未检测到明显声音
```

**解决**:
1. 检查 Windows 麦克风权限
2. 对着麦克风大声说话
3. 检查麦克风是否静音
4. 指定设备索引:
   ```python
   # 在脚本第 145 行附近
   mic = MicrophoneDevice(device_index=0)
   ```

---

### 问题 4: 测试卡住

**解决**:
1. 按 **Ctrl+C** 中断当前测试
2. 查看错误信息
3. 重新运行测试
