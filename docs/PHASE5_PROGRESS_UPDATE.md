## 🎓 使用示例

### 调用导航技能

```python
from zulong.expert_skills import SkillPool, NavigationSkill

# 初始化技能池
skill_pool = SkillPool()

# 注册导航技能
skill_pool.register_skill(
    skill_type="navigation",
    factory_func=lambda: NavigationSkill(skill_id="nav_001"),
    gpu_memory_mb=0,
    cpu_memory_mb=256,
    priority=5
)

# 调用技能
result = skill_pool.call_skill(
    skill_type="navigation",
    method="plan_path",
    start=(0.0, 0.0),
    goal=(2.0, 2.0)
)

print(f"规划路径：{result.result['path']}")
print(f"总距离：{result.result['distance']:.2f}m")
```

### 多技能工作流

```python
# 定义工作流
workflow = [
    {
        "skill": "vision",
        "method": "detect_objects",
        "args": [None],
        "critical": True
    },
    {
        "skill": "navigation",
        "method": "plan_path",
        "args": [(0.0, 0.0), (1.5, 1.5)],
        "critical": True
    }
]

# 执行工作流
results = skill_pool.orchestrate(workflow)

for i, result in enumerate(results):
    print(f"步骤 {i+1}: {'成功' if result.success else '失败'}")
```
