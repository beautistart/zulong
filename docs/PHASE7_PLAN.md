## 📝 风险与应对

### 风险 1: 显存不足

**现象**: RTX 3060 6GB 无法同时加载三个模型

**应对**:
- 优化量化策略（4bit → 2bit）
- 增强热切换机制
- 强制 CPU 运行部分模型

### 风险 2: 推理速度不达标

**现象**: InternVL 推理速度 >2s

**应对**:
- 调研更轻量模型
- 模型蒸馏/剪枝
- 异步推理优化

### 风险 3: 文档工作量大

**现象**: API 文档和示例耗时超出预期

**应对**:
- 使用自动化工具生成 API 文档
- 优先完成核心示例
- 分阶段完善
