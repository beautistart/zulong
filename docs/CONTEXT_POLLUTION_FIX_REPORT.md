## 📋 问题诊断 (来自文档分析)

### 核心问题

1. **L2 模型"发疯" (Instruction Leakage)**
   - **现象**: 输出大量重复的规则复述 (`5. 必须包含... 6. 必须包含...`)
   - **原因**: Prompt 结构不清晰，模型把指令当成回答内容复述

2. **视觉感知与对话逻辑割裂**
   - **现象**: InternVL 正确识别了内容，但 L2 回复忽略视觉信息
   - **原因**: 长 Prompt 中发生"注意力迷失",忽略中间的视觉描述

3. **视觉信息截断**
   - **现象**: 视觉描述以"..."或"了"结尾，信息不完整
   - **原因**: 未进行句子补全处理

### 修复 2: 异常输出拦截器

**文件**: `zulong/utils/text_cleaner.py`  
**位置**: `clean_text_for_tts()` 函数开头

**检测模式**:

```python
anomaly_patterns = [
    (r"必须包含.*关键要素", "规则复述"),
    (r"约束条件.*", "规则复述"),
    (r"版本一：.*版本二：", "多版本输出"),
    (r"^(\d+\..*?\n){3,}", "连续数字列表"),
    (r"5\..*必须.*6\..*必须", "复述约束条件"),
    (r"^\d+\..*必须", "数字列表 + 必须"),
]
```

**处理策略**:
- **策略 A**: 提取正常内容 (查找包含人称代词或动词的句子)
- **策略 B**: 如果没有正常句子，返回兜底回复

**代码示例**:
```python
# 查找正常句子
normal_patterns = [r'[你我他].*在.*', r'我看到.*', r'你正在.*', r'根据.*']
for np in normal_patterns:
    match = re.search(np, cleaned)
    if match:
        # 提取匹配的句子
        start = match.start()
        end = cleaned.find('.', start)
        if end != -1:
            result = cleaned[start:end+1].strip()
            if len(result) > 10:
                return result  # 返回正常内容

# 没有正常句子，返回兜底回复
return "我刚才有点走神，请再说一遍您的问题好吗？"
```

### 修复 3: 视觉信息补全

**文件**: `zulong/l2/inference_engine.py`  
**位置**: 新增 `_complete_visual_sentence()` 方法

**逻辑**:
```python
def _complete_visual_sentence(self, text: str) -> str:
    """补全视觉描述句子 (避免截断)"""
    # 检测未完成的句子
    incomplete_endings = ['...', '了', '着', '，', ',', '在', '是']
    
    for ending in incomplete_endings:
        if text.endswith(ending):
            last_period = text.rfind('.')
            if last_period != -1 and last_period > 10:
                # 截断到最后一个完整句号
                text = text[:last_period + 1]
            else:
                # 没有完整句号，添加标记
                text = text + " (场景描述)"
            break
    
    return text
```

**效果**:
- ✅ 避免截断的视觉描述传递给 L2
- ✅ 保证视觉信息完整性
